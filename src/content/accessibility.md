# Barrierefreiheitserklärung

Pipelinr ist bemüht, seine Website barrierefrei zu gestalten und möglichst vielen Menschen zugänglich zu machen.

Diese Website wurde im Rahmen eines Studienprojekts erstellt und stellt ein fiktives Softwareprodukt dar.

## Konformitätsstatus

Unser Ziel ist es, die Barrierefreiheit bestmöglich zu erreichen. Dazu gehören folgende Richtlinien und Gesetzte:

- Bundesgesetz zur Barrierefreiheitsanforderung (Barrierefreiheitsgesetz - BaFG) 
- die Richtlinie (EU) 2019/882 des Europäischen Parlaments
- und die WCAG 2.2 (Web Content Accessibility Guidelines)

## Maßnahmen zur Barrierefreiheit

Folgende Maßnahmen wurden ergriffen:

- Verwendung semantischer HTML-Elemente
- Ausreichende Farbkontraste gemäß WCAG 2.2 Erfolgskriterium 1.4.3 
- Beschriftungen für Links und Schaltflächen
- Unterstriche für Links in Textbereichen zur visuellen Unterscheidung
- Hervorhebung von aktiven Textfeldern

## Technische Spezifikationen

Die Barrierefreiheit dieser Website beruht auf folgenden Technologien:

- HTML
- CSS
- JavaScript (React)

## Bekannte Einschränkungen

Folgende Inhalte oder Funktionen sind derzeit nicht vollständig barrierefrei:

**Codeblöcke:** Die Syntaxhervorhebung in den YAML-Beispielen wird nicht durch Screenreader übermittelt. Der Text ist jedoch vollständig lesbar.

## Durchsetzungsverfahren

Beschwerde an die Österreichische Forschungsförderungsgesellschaft mbH (FFG) wenden.
Nutzen Sie dazu das elektronische Kontaktformular erreichbar unter 
[www.digitalbarrierefrei.at/de/beschwerdestelle/kontaktformular-beschwerdestelle](https://www.digitalbarrierefrei.at/de/beschwerdestelle/kontaktformular-beschwerdestelle).

Sofern die Beschwerde berechtigt ist, hat die FFG die Möglichkeit, dem Bund oder betroffenen Rechtsträgern Handlungsempfehlungen vorzuschlagen. 

Weitere Informationen finden Sie auf der Website der Österreichischen Forschungsförderungsgesellschaft mbH (FFG) unter [www.ffg.at/barrierefreiheit/beschwerdestelle](https://www.ffg.at/barrierefreiheit/beschwerdestelle).
